Create Project

Here we start to create a simple cocos2dx-win8 project. Before we start you should to read Installation

Create Project

  1. Open vs2011, Click menu: File->New->Project..

  2. Choose Templates->Visual C++-> Windows Metro style, select the Cocos2d template to create project like followed snapshot:
    Here,we create at "d:\Projects\"

  3. Click Ok, then you may see followed snapshot, this seems a bug of visual studio, so we just ignore them and click OK. After all close Visual Studio.

  4. Copy "d:\cocos2dx-win8\CreateProject.bat" to "d:\Projects\helloworld-cocos2d"

  5. Edit CreateProject.bat,set appropriate variable

    set PROJECT_NAME=helloworld-cocos2d
set COCOS2D_DIR=d:\cocos2dx-win8
set PROJECT_DIR=d:\Projects\helloworld-cocos2d

  6. Run CreateProject.bat, it will copy some src code files and some resources into "d:Projectshelloworld-cocos2d". After this, your will see dir below

  7. Open "d:\Projects\helloworld-cocos2d\helloworld-cocos2d.sln" and add "d:\Projects\helloworld-cocos2d\helloworld-cocos2d\helloworld-cocos2d.vcxproj" into.

  8. Done!

Configure Project

If you encounter a problem that can not be deployed, can be set as follows

  • Open Package.appxmanifest -> Packageing -> Choose Certificate -> Configure Certificate.. -> Create test Certificate.. -> Click Ok

Build and Run

Set Local Debugger, Native Only, Debug/Release Win32
