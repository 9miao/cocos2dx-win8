What is not implemented?

As most features are Implemented in cocos2dx-win8, but there are still serval features that not implemented.

  • CCAccelerometer
  • Text input
  • Chipmunk pyhics engine is not supported now since it cant pass the compiler.

Where is libxml?

Since windows8 metro only support a subset of win32 API and the libxml contains some win32 API that not be supported(can not pass the ACK), so we use tinyxml instead.
